# 📚 Kotoby — Library Management System

Kotoby is a simple and modern **Library Management System** built with Django.  
It allows admins to manage books efficiently with a clean and responsive UI.

---

## 🚀 Features

- ➕ Add new books
- ✏️ Edit existing books
- 🗑 Delete books with confirmation
- 📖 View book details
- 🔐 Admin authentication (login required)
- 🖼 Upload book cover images
- 📂 Categorize books
- 📊 Admin dashboard

---

## 🛠️ Tech Stack

- Python
- Django
- HTML5 / CSS3
- Bootstrap (optional if used)
- SQLite (default database)
